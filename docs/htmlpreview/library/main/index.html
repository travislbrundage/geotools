The gt-main module provides default implementations for the remaining gt-api and gt-opengis interfaces (Filter, Style, Feature etc...) and enough glue code to make creating an application possible (various builders and utility classes).

The gt-main module is responsible for:

  • Default implementation gt-opengis interfaces for Feature, FeatureType and Filter and Style
  • Default set of gt-opengis Converters supporting basic Java types
  • Default set of gt-opengis Functions for working with spatial data
  • Helper classes for your own application development such as DataUtilities and SimpleFeatureTypeBuilder
  • Abstract classes to help implementors of gt-api DataStore

Maven:

<dependency>
  <groupId>org.geotools</groupId>
  <artifactId>gt-main</artifactId>
  <version>${geotools.version}</version>
</dependency>
